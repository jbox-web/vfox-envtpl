# vfox-envtpl

[![CI](https://github.com/jbox-web/vfox-envtpl/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-envtpl/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [envtpl](https://github.com/jbox-web/envtpl)

## Install

```
mise plugin install envtpl https://github.com/jbox-web/vfox-envtpl.git
```
